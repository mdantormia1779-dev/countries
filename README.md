# 🌍 Countries Info App

A React app that fetches country data from an API and displays it using Suspense for smooth loading.

---

## 🚀 Live Demo

🔗 Live Site: https://countryapiprojectfreeapi.netlify.app/
🔗 GitHub Repo: https://github.com/mdantormia1779-dev/countries.git

---

## 🎯 Project Overview

This project is a **Countries Info App** that:

- Fetches country data from the API  
- Displays a list of countries dynamically  
- Uses **React Suspense** for async data loading  

---

## ✨ Features

- 🔹 Fetch data from API (`fetch`)  
- 🔹 Display country list dynamically  
- 🔹 Suspense fallback during loading  
- 🔹 Lightweight and simple React project  


---

## ⚙️ How to Use

### 1️⃣ Clone the Project

```bash
git clone https://github.com/mdantormia1779-dev/countries.git
cd countries
---


2️⃣ Install Dependencies
npm install
3️⃣ Run the Project
npm start
The app will automatically open in your default browser at http://localhost:3000
Data will be fetched from the API and displayed
🔄 API Endpoint
Get all countries:
https://openapi.programming-hero.com/api/all
⚠️ Notes
Ensure you have internet connection to fetch data from the API
Suspense fallback shows a loading message while data is loading
